# QR-awesomeness
A set of codes dedicated to QR factorization:
  1. Householder, Givens, Gram Schmidt QR
  2. Businger and Golub's QR with column pivoting
  3. Block QR with column pivoting
  4. Recursive Block QR with column pivoting
  5. LAPACK's DGEQP3
